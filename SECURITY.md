# Security policy

## Advisories

Security advisories are posted [on GitHub](https://github.com/robinlahtinen/clj-artnet/security/advisories).

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅         |
| < 1.0.0 | ❌         |

## Reporting a vulnerability

Report security vulnerabilities [via GitHub](https://github.com/robinlahtinen/clj-artnet/security/advisories).

Report vulnerabilities in third-party dependencies to their maintainers.

## Disclosure policy

When you report a security vulnerability, the maintainers follow this process:

1. Confirm the issue and determine the affected versions.
2. Audit the codebase for similar issues.
3. Prepare and release fixes as quickly as possible.

## Comments on this policy

If you have suggestions for improving this process, submit
a [pull request](https://github.com/robinlahtinen/clj-artnet/pulls) or open
a [discussion](https://github.com/robinlahtinen/clj-artnet/discussions).
